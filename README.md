# MealDeal App
## Overview
#### MealDeal is an app to explore recipes based on category, areas, ingredients or by name.

## Tech Stack
#### 1. React js, 2. Typescript

## Dependencies
#### 1. Material UI for styling
#### 2. Axios for Data fetching
